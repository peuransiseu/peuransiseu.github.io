## Features

- **Responsive Design**: Mobile-first approach with Bootstrap 5
- **Single Page Application**: Smooth scrolling navigation
- **Portfolio Slider**: Swiper.js integration for portfolio showcase
- **Text Animations**: AOS (Animate On Scroll) library for text effects
- **Lightbox Gallery**: Chocolat.js for image viewing
- **Skills Progress Bars**: Animated skill charts
- **Contact Form**: Functional contact form with validation
- **Mobile Menu**: Hamburger menu for mobile devices

## Sections

1. **Home**: Hero section with banner image and introduction
2. **About**: Education, experience, interests, and references
3. **Experience**: Personal information and skills chart
4. **Portfolio**: Portfolio slider with project showcase
5. **Contact**: FAQ accordion and contact form

## Technologies Used

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- JavaScript (ES6+)
- Bootstrap 5.3.1
- Swiper.js (Portfolio slider)
- AOS (Animate On Scroll)
- Chocolat.js (Lightbox)
- Google Fonts (Roboto)

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Setup

1. Open `index.html` in a web browser
2. All dependencies are loaded via CDN
3. Replace placeholder images with actual images
4. Customize content as needed
